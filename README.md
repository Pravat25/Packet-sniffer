![Python](https://img.shields.io/badge/Python-3.x-blue)
![Cybersecurity](https://img.shields.io/badge/Tool-Packet%20Sniffer-red)

## Overview

Packet Sniffer is a Python tool that captures network packets and displays basic information such as source and destination IP addresses.

This project demonstrates fundamental **network traffic analysis**, a key concept in cybersecurity.

## Features

- Capture network packets
- Display source and destination IP addresses
- Simple command-line interface

## Technologies Used

- Python
- Scapy library

## Installation

Clone the repository

git clone https://github.com/Pravat25/packet-sniffer.git

Navigate into folder

cd packet-sniffer

Install dependencies

pip install -r requirements.txt

## How to Run

Run the script

python sniffer.py
